1.title
Web Page Layout Basic With Box Model Concept （CSS语言的盒状模型概念和排版基础）
2.introduction
盒状模型概念对于理解CSS自动排版功能至关重要，Web页作为矩形区域，其内部元素的外观本质上也是矩形，各个元素按照自己盒子模型的设定，与周边元素互相影响，实现了Web页的自动排版。
本课我们仍以div这个抽象块级元素，结合p段落来探索CSS盒状模型内的一些重要概念，包括外边距margin，内边距padding，以及相关CSS语法。

3.keyword
box model ，div ，margin   #id ，class ，width  ，em ，width = border + padding + content 。
