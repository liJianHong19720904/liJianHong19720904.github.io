1.title
Browser For Learning Web Programming  （学习 “Web编程基础” 浏览器环境）

2.introduction
理解计算机、操作系统、浏览器、代码编辑器的层次关系，能阐述本地HTML文件是如何运行的。能在计算机（Windows平台）上使用和安装代码编辑器、浏览器。
使用代码编辑器手写几个本地HTML文件，体验HTML标记的基本编写方法，能用浏览器打开的本地HTML文件，运行HTML文件。

3.keyword
windows/Linux/MacOS/Android ，browser ，Firefox / Chrome /Opera /Safari /Internet Explorer /Microsoft Edge ，HTML document 。
