Operating system and applications

An operating system (OS) is a piece of software that manages a computerâ€™s hardware and software resources, and makes it easier for us to use them. There are many types of OS, and they can be used for different purposes.

An application is a type of software that allows its user to perform a specific task. The OS provides a framework for these applications to run.

The operating system uses applications called utilities, which allow users to manage their computer, its devices, and its programs. There are many different utility programs, and they vary across operating systems. Users can access these via a special menu or control panel.
