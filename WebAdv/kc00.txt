2.introduction
欢迎来到"Learn Programming In Web Front"课程首页！把鼠标移动到最上面英文标题，会有中文互动反馈。本课程我们以Web和前端的编程学习开发为背景，通过实践和理论结合，尝试用代码来探索计算机、网络、软件、Web的概念和道理，掌握编程的实践方法和理解理论，从而达到能写出更高质量代码（没做最高）的目标。
修习本站的内容一般需要有高级程序语言（首选C语言）学习经历。对于一般的计算机高级语言基本语法都还不熟悉的编程初学者，则需要预先入门一门简单的高级语言，编一些小案例打好通用语法基础，再学习本站的内容也不迟。
本站内容一共由20个大课组成，每一大课都规范地按“内容简介”、“本课关键字”、“实践案例”、“理论阅读”四个环节组成。
其中“实践案例”是本课程的学习重点，这些实践案例都采用逐步提升难度的方式设计，学习者可按自己的代码功力加以选择， 不同学习者通过写这种由浅入深的代码案例，就会较容易找到实践编程中自己个性的问题，克服挫折，乃至激发自己创作代码的兴趣，找到自己的乐趣。
每一大课内容基本围绕一个主题，自成体系，对于有兴趣的学习者可以继续深入探索。每课还在最后的阅读区展示一些不错的英文文本，源于各种经典阐述，反复阅读可以提升英文能力，同时理解也可以提高认知，顺便修炼我们编写代码的内功。多阅读经典能让开发者站在更高的视角，训练思维，提高分析问题和解决问题的能力。

This beginner's course is written for complete beginners to the world of website code, who don't know what to learn first , and just need a single poiont that will lead them in the right direction with confidence. This series of lessons will guide you through all the basics of web development , so you can get up to speed relatively quickly, learning just enough to not feel lost and overwhelmed, and get comfortable with the area in general. We try to give you important background information of how the web works , we are actually interested in telling you how the bricks are made , and we're not just want you to start building the house .
This course is also for front-end web developers. That means we teach you how the front, or client side of the web works, not the processing, or back side. We teach you how to create webpages with languages like HTML, CSS, and JavaScript (which we'll code mainly). You'll be free to explore on your own and experiment with the front-end code that make the web work.In short, you'll be confident and comfortable with the usage and coding of web technologies like HTML, CSS, and JavaScript. This course is not for back-end web developers, who are responsible for hosting web servers  .  Also, this course specifically won't go in much details of other interesting and efficient web technologies like SVGs, or Canvas APIs. You can learn them easily after accomplishing this course with some good materials.
